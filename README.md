# ALE DEV — Portfolio

Portfolio personal de **Alexis Villalba**, desarrollador web fullstack de Rafaela, Santa Fe, Argentina.

🔗 **Live:** [aledevweb.com](https://aledevweb.com)

---

## Tecnologías utilizadas

- **HTML5** — estructura semántica
- **CSS3** — animaciones, glassmorphism, diseño responsive
- **JavaScript (Vanilla)** — lógica de animaciones e interactividad
- **Three.js** — animaciones 3D (partículas hero, esfera about)
- **GSAP** — animaciones de scroll
- **Google Fonts** — Archivo + Space Grotesk

## Características

- Fondo 3D con campo de 2000 partículas y formas geométricas flotantes
- Esfera 3D animada con anillos orbitales
- Cursor personalizado con efecto de seguimiento
- Typing animation con roles rotativos
- Scroll reveal en todas las secciones
- Barras de skills animadas al entrar en viewport
- Navbar glassmorphism flotante
- Diseño responsive (mobile, tablet, desktop)
- Indicador de scroll animado

## Secciones

1. **Hero** — presentación con partículas 3D
2. **Sobre mí** — perfil y stack tecnológico
3. **Skills** — tecnologías con nivel de dominio
4. **Servicios** — servicios que ofrezco
5. **Formación** — educación y certificaciones Coderhouse
6. **Proyectos** — trabajos destacados
7. **Testimoniales** — clientes satisfechos
8. **Contacto** — formulario + redes sociales
